# Cómo ejecutar JavaScript

Para ejecutar código JavaScript puedes usar el **navegador** (con la consola), o **Node.js** en la línea de comandos. En el artículo original se explican ambos métodos y cómo configurar un editor.

*Resumen del artículo:* 
- Abrir la consola del navegador (Ctrl+Shift+I) y escribir `console.log('hola')`.
- Instalar Node desde https://nodejs.org y ejecutar `node archivo.js`.
- Usar VS Code con la extensión **Run Code** para ejecutar rápidamente.
