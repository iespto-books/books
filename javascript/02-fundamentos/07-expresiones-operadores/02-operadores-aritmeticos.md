# Operadores aritméticos en JavaScript

Los operadores aritméticos permiten realizar cálculos matemáticos básicos. Los más comunes son `+`, `-`, `*`, `/`, `%` y `**` (exponenciación). Cada uno sigue una precedencia definida.

```js
let a = 5 + 3; // 8
let b = a * 2; // 16
let c = b ** 2; // 256
```

Para más ejemplos y detalle, consulta el artículo original en Luis Llamas.
