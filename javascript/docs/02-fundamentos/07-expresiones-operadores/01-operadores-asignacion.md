# Operadores de asignación en JavaScript

Los operadores de asignación combinan una operación con la asignación de un valor a una variable.

| Operador | Significado |
|----------|--------------|
| `=` | Asignación simple |
| `+=` | Suma y asigna |
| `-=` | Resta y asigna |
| `*=` | Multiplica y asigna |
| `/=` | Divide y asigna |
| `%=` | Módulo y asigna |
| `**=` | Exponenciación y asigna |

Ejemplo:
```js
let a = 5;
a += 3; // a = 8
```
