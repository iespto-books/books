# ¿Qué es JavaScript?

JavaScript es un lenguaje de programación **interpretado**, **dinámico** y **multiplataforma** que permite crear contenido interactivo en la web. Fue creado inicialmente para añadir comportamiento a las páginas HTML, pero hoy en día se usa también en servidores (Node.js), aplicaciones móviles (React Native) y mucho más.

> **Dato curioso**: el nombre “JavaScript” surgió por motivos de marketing; no tiene relación directa con Java.
