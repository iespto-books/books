# Historia y evolución

| Año | Hito |
|-----|------|
| 1995 | Brendan Eich crea *Mocha*, luego renombrado a **LiveScript** y finalmente a **JavaScript** para Netscape. |
| 1996 | Microsoft lanza **JScript** (compatible con ECMAScript 1). |
| 1997 | **ECMAScript 1** (ES1) estandarizado por ECMA‑International. |
| 1999 | **ES3** introduce `try/catch`, `strict mode`, expresiones regulares mejoradas. |
| 2005 | **Ajax** populariza JavaScript en la web. |
| 2009 | **Node.js** permite ejecutar JS fuera del navegador. |
| 2015 | **ES6 / ES2015** (let/const, clases, módulos, arrow functions, promises, …) marca una gran revolución. |
| 2016‑2025 | Lanzamientos anuales: ES2016, ES2017, … ES2024 añadiendo async/await, optional chaining, etc. |
