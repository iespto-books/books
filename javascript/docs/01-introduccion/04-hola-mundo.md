# Cómo ejecutar JavaScript

Para ejecutar código JavaScript puedes usar el **navegador** (con la consola), o **Node.js** en la línea de comandos. 

*Resumen del artículo:* 
- Abrir la consola del navegador (Ctrl+Shift+I) y escribir `console.log('hola mundo')`.
- Instalar Node desde https://nodejs.org y ejecutar `node archivo.js`.
- Usar VS Code con la extensión **Run Code** para ejecutar rápidamente.
