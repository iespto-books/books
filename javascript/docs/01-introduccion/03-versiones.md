# Versiones de JavaScript (ECMAScript)

| Versión | Año | Principales novedades |
|---------|-----|------------------------|
| ES5 (2009) | 2009 | `strict mode`, `Array.forEach`, `JSON` |
| ES6 / ES2015 | 2015 | `let/const`, clases, módulos, arrow functions, promises, spread, template literals |
| ES2016 | 2016 | Operador exponenciación (`**`), `Array.includes` |
| ES2017 | 2017 | `async/await`, `Object.entries/values` |
| ES2018 | 2018 | Rest/Spread para objetos, mejoras RegExp |
| ES2019 | 2019 | `Array.flat`, `Object.fromEntries`, `optional catch binding` |
| ES2020 | 2020 | Nullish coalescing (`??`), optional chaining (`?.`) |
| ES2021 | 2021 | Logical assignment (`&&=`, `||=`, `??=`) |
| ES2022 | 2022 | `class` fields, `Array.at()` |
| ES2023 | 2023 | `Array.findLast`, propuestas en etapa 4 |
